Servicios

  • Desarrollo Web

    Las necesidades de la empresa expresadas en aplicaciones que permitan la gestión de la información de manera eficiente y rápida, es cada dia mas apremiante.

    En Aliter ofrecemos el servicio de Desarrollo de aplicativos bajo las plataformas .Net y Java haciendo uso de metodologia Scrum para la obtencion de resultados predecibles y rápidos.

    Las bases de datos mas utilizadas para ello son Oracle y SQL Server. El uso de micro servicios para la comumicación con los cores de la empresa también son proveidos.

  • Aplicaciones Móviles

    Ofrecemos el desarrollo de aplicaciones móviles para empresas como una manera efectiva de gestionar la información de la empresa por parte de la fuerza de venta y administrativos que necesitan estar en el campo con equipos ligeros como tables.

    Se estima que hay un 60% de posibilidad compra de productos por medio de aplicaciones móviles y este porcentaje va en aumento.

    Elaboramos apps nativas para equipos Android bajo la plataforma Android Studio y Angular 7.

  • Aseguramiento Calidad

    Elaboramos planes de Pruebas, casos de Prueba y SLAs para las aplicaciones desarrolladas por nosotros o terceros.

    De esta manera llevamos a cabo un proceso formal de validación del correcto funcionamiento de las aplicaciones asegurando que se satisfaga los requerimientos funcionales establecidos para el mismo.
